# Math_writings
These are expositions on some mathematical topics. 

fft_bit_reversal - This is an explanation of how the Danielson-Lanczos lemma works in calculating the Fast Fourier Transform through detailed but short calculations. There are also a few words on the bit reversal part of an FFT algorithm.

more_bit_reversal - This is a deeper dive into binary palindromesand their distribution.

inverse_functions - Finds the derivative of an inverse function $f^{-1}(x)$ using $f(x)$ and uses that to find the functional form of $f^{-1}(x)$.
